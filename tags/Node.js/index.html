通过nvm安装Node.js

2022年3月18日
| Node.js
有段时间没有使用node.js了，今天发现电脑上环境已经破坏，那就只能重安装了。 前提 安装前先将本地已有的node.js卸载干净。 下载 nvm 首先下载最新版nvm-windows安装。 安装时候需要选择一个nvm安装目录，一个node.js软连接目录。安装过程会自动将这两个目录加入环境变量……
