Spring Cloud入门教程(01)
By: Jonathan Lai |  2022-03-20  | spring, spring-cloud, java
教程由来 都是Java开发十多年的老鸟了，微服务一套基本都用的是 Dubbo 及相关系列，有些东西都是自研，但现实是新人还有周边朋友都在用 Spring Cloud 一套，说是包含了很多微服务用到的东西，使用起来很方便快捷，搭建项目也很舒爽，因此，自学看看呗，到底爽点在哪里。 关于教程 也不算教程吧，就是记录一下自己学习的过程，方便之后回头来看看。 至于内容，也没有什么具体的安排，根据自己学习过程来吧，学到哪里算哪里了，之后如果有空，再系统的整理吧。 关于 Spring Cloud Spring Cloud 为开发者提供了工具来快速构建分布式系统中的一些常见模式（例如配置管理、服务发现、断路器、智能路由、微代理、控制总线、一次性令牌、全局锁、领导选举、分布式会话、集群状态）。分布式系统的协调导致了样板模式，使用 Spring Cloud 开发人员可以快速建立实现这些模式的服务和应用程序。它们在任何分布式环境中都能很好地工作，包括开发人员自己的笔记本电脑、裸机数据中心以及 Cloud Foundry 等托管平台。 不废话，我们直接开始，至于具体的概念，后续遇到会单独提出来解释。 入门项目 第一步先生成一个新的 Spring Boot 项目。 网页打开 https://start.spring.io/ 按照要求填写，如下图： 然后导出（CTRL+回车键），下载到本地，并解压。 为了后续教程的支持，我这里将项目结构做一些调整，调整结果如下： 主要调整的地方： 1、将原先的tutorials-parent项目调整成父级项目，packaging类型为pom。 2、新建子文件夹tutorials-01作为子项目，也就是教程01的项目，以后依次每次教程都会新建一个子项目来处理。 主要调整就是这两点，具体调整代码可能有点多，需要的话去 Github 拉源码看看。 调整完成之后，使得springboot项目运行成功即可： 今天就先到这儿。 学习过程中用到的包、框架等等，我尽量都用最新的。 作为一个程序猿，保持最新，把握技术命脉是必备技能之一。
